Accessible Weather and Shelter Finder
- Accessible web application for blind and visually impaired users.
- Provides real-time weather updates, severe weather alerts, and nearest shelter locations.
- Accessibility support includes keyboard navigation, high-contrast UI, shortcut keys, and text-to-speech.

Beyond Sight - Audible Graphs for the Visually Impaired
- AI accessibility system converting charts and images into speech for visually impaired users using CNNs, OCR, and TTS.
- Presented at ACM W4A 2024.
- Achieved 92% accuracy in chart classification and improved accessibility by 90%.
